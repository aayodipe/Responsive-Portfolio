Warratie Log
Full Stack
HTML5 CSS3 JavaScript MYSQL NODEJS EXPRESS

Warrantie Log is an app that helps the user to keep track of their item's warranties. Users can add, update or delete items. If the items expiration date is less than a month, the app will move the item to Expire-soon section, then send a reminder to the user to inform them about the item expiration date. The user can either renew the expiration date or do nothing. If the Item expiration date is renewed, the user can then update the expiration date and if nothing is done, the app will delete the items once the expiration date elapses.
