Jeliteinstallationsllc is a website for a production website developed for a client on their request. It's a five page website that showcase the information of Jelite Installation LLC.This website use Boostraps, Jquery and Vanilla Javascript. It's highly mobile responsive.

All Things Travel
All things Travel is a Travel Site Web App. It requires the user to fill in it's travel information(Origin Cities, Destination City,Dates and accessories.The date should not exceed 14 days interval). The information will pass through all of the APIs and populate on the page in one view with links to the Kayak website for more flight information or the website/menu of popular restuarants in the destination city. This is a group project. The application uses three APIs; the Kayak, Zomato, and Weatherbit APIs. Our design was a very clean look with a carousel of images to iconic locations with inputs for the users departure city, destination, when they are leaving and returning, how many travels, and number of bags.
Train Management
Train Management is a Train Schedule Web Application. This website uses HTML5, CSS3, Bootstrap,Jquery and Vanilla Javascript for its frontend and Firebase for it's data storage. It also uses Moment.js Libiary for time convertion and ekko lightbox for image galary. This application is mobile responsive. Train Management takes and validate use input. If user the user input is valid, it added the information to the Schedule table else it inform the user to input correct information.
